Neural Networks

Styled Page

Neural Networks is a form of unsupervised learning that was first developed in 1944 by Warren McCullough and Walter Pitts. Neuaral networks incorporates the means of doing machine learning where the computer learns to perform a specific task in the training data and output visual patterns or images that are correlated with the input data. Neural networks are more prominent in layers of nodes in feedforward processes, where the data moves in one direction. Nodes can be associated with several nodes within another layer that are hidden before or after the movement of the data. In this section, we will expand upon the process of feedforward process, neural network architecture, and affiliated processes.

To begin with, we will be discussing the structure of a feedforward network. In the feed forward network, we begin with an input layer that contains neurons (nodes) in which the data pass through and received by neurons in the hidden layers that has output nodes that has some type of association. This association is triggered by particular patterns that is incorporated in first the neuron which allows it to associate itself with the corresponding neuron in the forward hidden layer that provides similar features.

Withih the hidden layer, the neurons are assiges weights to the inputs and pass through the layer. Inside of the neurons, there is a number ranging from 0 to 1 to represent the activation function. The activation function is considered the sigmoid function which helps maintain the weighted sum by reinforcing the constraints of the range to be between 0 and 1. By updating the function, we incorporate the bias term which help manages the meaningfulness of the neuron activeness. The bias term is used as an offset parameter with weights to control the activation or "influence" of each input neuron on the output. Each weight has an associated bias.
